## 6、从库crash-safe

Crash-safe意味着在MySQL或者OS宕机以后，依然可以直接恢复MySQL主从同步，而不是利用备份来重建从库。
为了确保crash-safe，需要确保以下几点：

- 只使用InnoDB存储引擎

- 在5.6版本以后，需要设置以下选项
   
   ```bash
  relay_log_info_repository=TABLE;
  relay_log_recovery=1;
   ```
